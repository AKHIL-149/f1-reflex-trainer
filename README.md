# 🏎️ F1 Reaction Training Drill

Train your reflexes like a Formula 1 champion driver with this interactive web application featuring two specialized reaction training games.

## 🎮 Games

### Game 1: Left/Right Buzzer
- Press the correct buzzer (left or right) when called
- Track attempts, correct/wrong/missed responses
- Monitor reaction times and averages
- Three difficulty levels with variable delays

### Game 2: Ball Drop Catch
- Physical training with hand-specific callouts (left, right, or both hands)
- Drop and catch a real ball based on audio cues
- Customizable delay times for each difficulty level
- "Both hands" callouts increase with difficulty:
  - **Easy**: Left/Right only
  - **Medium**: 10% both hands
  - **Hard**: 30% both hands

## ✨ Features

- **Multiple Difficulty Levels**: Easy (3-5s), Medium (2-4s), Hard (1-3s)
- **Voice Callouts**: Text-to-speech audio cues
- **Audio Feedback**: Different beep frequencies for different actions
- **Mobile Responsive**: Works on all devices with touch support
- **Customizable Timing**: Adjust delay times for Game 2 based on your preference
- **Real-time Stats**: Track your performance with live statistics
- **F1-Themed Design**: Sleek red and black racing-inspired interface

## 💻 Local Development

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/AKHIL-149/f1-reflex-trainer.git
cd f1-reflex-trainer
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open your browser and navigate to:
```
http://localhost:3000
```

**Alternative**: Simply open `index.html` directly in your browser (no server needed for local testing).

## 📱 Controls

### Game 1: Left/Right Buzzer
- **Click/Touch**: Press the buzzers
- **Keyboard**: Arrow keys or A/D keys

### Game 2: Ball Drop Catch
- **Click/Touch**: Tap the green zone
- **Keyboard**: Spacebar or Enter

## 🎯 Training Tips

1. **Start with Easy Mode**: Build confidence and understand the game mechanics
2. **Focus on Accuracy**: Speed comes with practice
3. **Use Proper Form**: In Game 2, ensure proper catching technique
4. **Consistent Practice**: Short, regular sessions yield better results
5. **Progressive Difficulty**: Gradually increase difficulty as you improve

## 📦 Deployment

### Heroku Deployment

1. Create a Heroku account and install the Heroku CLI

2. Login to Heroku:
```bash
heroku login
```

3. Create a new Heroku app:
```bash
heroku create your-app-name
```

4. Deploy:
```bash
git push heroku main
```

5. Open your app:
```bash
heroku open
```
